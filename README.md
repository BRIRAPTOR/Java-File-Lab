# Java File Lab

Proyecto en Java para practicar operaciones con archivos, incluyendo lectura, escritura, eliminación y conversión entre texto y binario.

También incluye ejemplos de manejo de imágenes y uso de streams para trabajar con diferentes tipos de archivos.

## Tecnologías
- Java
- IntelliJ IDEA

## Objetivo
Practicar el manejo de archivos en Java de forma clara, ordenada y funcional.
